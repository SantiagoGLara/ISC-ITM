## tipos de desarrollo
### desarrollo nativo
uso de lenguajes nativos java/kotlin para android, swift/objective-c para IOS. En entornos de desarrollo como android studio y Xcode. Cuenta con acceso completo a las API y arquitectura del dispositivo, ademas de un rendimiento optimo y mejor experiencia de usuario. Necesita que se mantengan bases de codigo separadas por plataforma.

### desarrollo hibrido
Hace uso de tecnologias web como HTML, CSS, JS con frameworks como Ionic+angular, Apache Cordova, Sencha, React native. Estas aplicaciones corren en un contenedor nativo llamado **WebView**(funge similar a un navegador pero de una sola pagina, es una apk que contiene solo la capacidad de renderizar el html, css y JS de una app), cuenta con un desarrollo rapido, economico pero con limitaciones en rendimiento y experiencia de usuario.

### desarrollo multiplataforma
Frameworks como react native, flutter, xamarin, KMP. Una base de codigo utilizable en varias plataformas, reduce costos y tiempo de desarrollo respecto al nativo pero tambien tiene posible peor rendimiento. El compilador genera el binario para ambas plataformas.
