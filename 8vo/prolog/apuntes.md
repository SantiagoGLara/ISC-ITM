## introduccion
el objetivo de la programacion logica funcional a pensar de forma abstracta, modelar problemas complejos, evitar calculos matematicos extensos y complejos.
### programacion funcional:

ejemplos
- recomendaciones de nueva musica: es una serie de pasos cada uno dependiente del anterior orientado a funciones que siempre llegan a un resultado

- calculo de nominas: serie de funciones donde

- generacion de reportes
### programacion logica
ejemplos
- sistema de diagnosticos medicos: a partir de reglas, arboles de decisiones y/o condiciones nos pueden aproximar a resultados

- programa para aprobar un credito: a partir de reglas, arboles de deciciones y/o condiciones se define si es apto o no para el credito
